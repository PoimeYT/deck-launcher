# deck-launcher
A Minecraft launcher with controller-friendly UI, built with Electron.

## Setup

```bash
npm install
npm start
```

## Controller / Steam Deck

Arrow keys navigate between elements, Enter/Space to select.
In Steam Deck game mode, map your controller to keyboard via Steam Input.
